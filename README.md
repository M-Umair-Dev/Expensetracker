Expensetracker 💸
A full-stack MERN (MongoDB, Express, React, Node.js) application designed to help users track their daily expenses efficiently.

🚀 Live Demo
You can view the live project here: https://expensetracker-tau-two.vercel.app/

🛠 Features
User Authentication: Secure signup and login functionality.

Expense Management: Easily add, view, and manage your daily spending.

Responsive Design: Optimized for both desktop and mobile views.

Data Persistence: All expense data is safely stored in a cloud MongoDB database.

⚙️ Technologies Used
Frontend: React, Axios

Backend: Node.js, Express

Database: MongoDB (Atlas)

Deployment: Render (Backend), Vercel (Frontend)
How to Run Locally
Clone the repository:

Bash
git clone https://github.com/M-Umair-Dev/Expensetracker.git
Install dependencies:

Bash
# In the root folder
npm install
cd backend && npm install
cd ../frontend && npm install
Configure Environment Variables:
Create a .env file in the backend folder and add your MONGO_URI, JWT_SECRET, and PORT.

Start the application:

Bash
# Run both concurrently (if configured) or separately
npm run dev
